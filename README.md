O doToList é uma aplicação web para gerenciamento de tarefas, desenvolvida com foco em simplicidade, clareza e aprendizado prático dos fundamentos do desenvolvimento front-end. O projeto permite ao usuário adicionar, visualizar e remover tarefas de forma dinâmica, utilizando apenas tecnologias essenciais da web.

Este repositório tem como objetivo consolidar conhecimentos em HTML, CSS e JavaScript, com ênfase na manipulação do DOM, lógica de programação e organização de código. É um projeto voltado para nível iniciante/júnior, ideal para demonstrar evolução técnica e entendimento dos conceitos básicos exigidos no mercado.

Funcionalidades

Adicionar novas tarefas à lista

Listar tarefas inseridas dinamicamente

Remover tarefas individualmente

Interface simples e objetiva

Atualização da tela sem recarregamento da página

Tecnologias Utilizadas

HTML5

CSS3

JavaScript (Vanilla JS)

Conceitos Praticados

Estruturação semântica de páginas HTML

Estilização básica com CSS

Manipulação do DOM

Eventos em JavaScript

Lógica de programação
